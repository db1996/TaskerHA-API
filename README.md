# TaskerHA - API

Backend API for [TaskerHA](https://github.com/db1996/TaskerHa) — a full Home Assistant integration for Tasker.

Handles install tracking (device pings and stats) for the TaskerHA Android app. Built with Fastify and better-sqlite3.
